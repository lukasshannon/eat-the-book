Vanilla Page-Turn Book

A single-file, zero-dependency Web Component inspired by a CodePen book flip demo. Click either side of the book, swipe across it, use the buttons, or focus it and press arrow keys.

The café opens between heartbeats, warm with candlelight and the smell of sugar browned in a copper pan.

A recipe card waits inside the notebook. Its ink shifts when the page turns, as if the book remembers another kitchen.

This component is plain JavaScript: no jQuery, no turn.js, no image dependency, and no build step.

Use <page-turn-book> anywhere a static HTML page can load custom elements.
